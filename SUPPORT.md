# Support

NotATechBro is an early open-source project.

## Best places to ask for help

- Open a GitHub issue for bugs, confusing output, or setup problems.
- Open a feature request for new command explanations or adapter ideas.
- Open a pull request if you already have an improvement.

## What to include

For setup or adapter issues, include:

- your OS
- Node.js version
- agent CLI name and version
- the hook payload or command you tested
- the NotATechBro output
- whether you used `notatechbro` or the `change-preview` alias

Please redact secrets, tokens, private paths, and sensitive file contents before posting.
