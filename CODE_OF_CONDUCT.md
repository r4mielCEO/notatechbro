# Code of Conduct

## Our pledge

We want NotATechBro to be a welcoming project for people with different levels of technical experience. We pledge to make participation a harassment-free experience for everyone.

## Our standards

Examples of behavior that helps the community:

- Being kind, patient, and direct.
- Explaining technical tradeoffs without condescension.
- Respecting that this project is designed for non-technical and new agentic-coding users too.
- Giving constructive feedback with examples.
- Welcoming forks, experiments, and different product directions.

Examples of unacceptable behavior:

- Harassment, insults, or personal attacks.
- Dismissing people because they are less technical.
- Sexualized language or imagery.
- Publishing private information without permission.
- Spam or repeated off-topic promotion.

## Enforcement

Maintainers may remove comments, issues, pull requests, or other contributions that violate this code of conduct. Maintainers may also temporarily or permanently ban contributors for behavior they consider harmful to the project or community.

## Reporting

If you see a problem, open a private report by contacting the maintainer through GitHub. Please include links, screenshots, or context when possible.

This code is adapted from common open-source community standards and kept intentionally short for this project.
