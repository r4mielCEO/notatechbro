FAQ

Questions before you get started.

Does it block dangerous commands?

No. This version explains what is likely to happen. It does not approve, deny, block, or change tool calls.

Does it send commands or paths to a server?

No. It runs on your computer. There are no cloud calls, accounts, telemetry, or required AI service while it explains a command.

What will I see before a command runs?

You see the command and a short explanation of what it is likely to do. Your agent still asks whether you want to continue.

Who is this for?

It is for people using agentic coding tools who want plain English before they approve a command, especially if shell syntax feels unfamiliar.

Does it work in the Claude or Codex desktop apps?

The current verified release supports Claude Code CLI and Codex CLI. We do not yet claim that the preview appears inside either desktop app.
