Get started

Get NotATechBro working in three steps.

Install it once, confirm it is ready, then connect the coding agent you use.

Before you start

You need Node.js 18 or newer, npm, and either Claude Code CLI or Codex CLI. You do not need to download this repository or create a NotATechBro account.

Already installed? Go straight to connecting your agent.Choose Claude or Codex

1. Install NotATechBro

Copy this command into your terminal:

npm install --global notatechbro

2. Check the installation

notatechbro --check

You should see:

NotATechBro 0.1.0 is installed and ready.
Example: This will run this project's tests.

3. Connect your coding agent

Choose the tool you use and copy its setup:

Connect Claude Code Connect Codex

Try a safe first command

Start a new agent session and ask it to run the project tests. When the agent proposes npm test, NotATechBro explains:

This will run this project's tests.

The agent still asks whether you want to continue. NotATechBro never chooses for you.

Advanced testing options

Try the installation without installing globally:

npx notatechbro@latest --check

Test a hook payload manually:

echo '{"tool_name":"Bash","tool_input":{"command":"git push"}}' | notatechbro

For systems that require structured output:

echo '{"tool_name":"Bash","tool_input":{"command":"git push"}}' | notatechbro --json

The older change-preview command remains available for existing setups.

Runs locally. No cloud calls, telemetry, or NotATechBro account required.
