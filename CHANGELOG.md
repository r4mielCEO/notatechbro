# Changelog

All notable changes to NotATechBro are documented here.

## 0.1.0 — 2026-07-09

First public release.

- Plain-English previews for common shell commands and file operations.
- Adapters for Claude Code, Codex CLI, Hermes, and generic hook payloads.
- Risk and confidence metadata through `--json`.
- Quiet mode for read-only file previews.
- Install checks, help, and version commands.
- Local-only deterministic rules with no network or mandatory LLM.
